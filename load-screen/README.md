# Load Screen

Short load-screen animation created in Blender, focusing on a smooth looping motion suitable for UI or game loading screens.

## Preview

![Load Screen Animation](./load.gif)

For higher quality, play the MP4 export: `load0000-0048.mp4`.

## Details

- Software: Blender (4.x)
- Type: Looping load-screen animation
- Render engine: Cycles or Eevee
- Key techniques: Keyframing, interpolation/easing, seamless looping

## Files in this folder

- `load-screen.blend` – main Blender file
- `load.gif` – exported looping preview
- `load0000-0048.mp4` – rendered animation video
