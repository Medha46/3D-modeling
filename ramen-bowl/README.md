# Ramen Bowl

Stylized 3D ramen bowl created in Blender, focusing on clean shapes, soft materials, and appealing lighting.

## Preview

![Ramen Bowl](./ramen-bowl.jpg)

## Details

- Software: Blender (4.x)
- Type: Still render
- Render engine: Cycles or Eevee
- Key aspects: Stylized food modeling, simple shaders, soft lighting, depth and composition

## Files in this folder

- `ramen-bowl.blend` – main Blender scene
- `ramen-bowl.jpg` – final rendered image
